Definition of Done (aby sa to dalo obhájiť)

Minimum (na známku / požiadavky)

  • DB schéma + import .sql
  • PDO prepared statements (žiadne string query skladanie)
  • Login + sessions + logout
  • CRUD nad entitou článok (min. create + list + edit + delete)
  • Output escaping (XSS) + základné validácie

Plus (bonus – ak bude čas)

  • CSRF tokeny pre všetky POST formuláre
  • Flash messages + error pages
  • Tagy/kategórie (M:N) + filter
  • Jednoduché audit logy (napr. „kto upravil článok“)
  • Hardening sessions (cookie flags, regenerate id)
