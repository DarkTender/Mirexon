SENTRIX Roadmap

Plán učenia + plán projektu (PHP 8 + MySQL, OOP, CRUD)

Tento web začína ako statická cyber security stránka (writeups + tools). Postupne sa zmení na školský projekt: OOP + CRUD nad článkami + login v čistom PHP bez frameworku a s databázou MySQL.

Milníky (high-level)

Writeups
Obsah, kategórie, navigácia, konzistentný dizajn
PHP 8 + MySQL
PDO, OOP štruktúra, CRUD pre články
Login & Security
sessions, password_hash, CSRF, output escaping

Obrázky môžeš neskôr vymeniť za vlastné (napr. screenshoty z logov, dashboardov, labu).

🧱Požiadavky projektu (checklist)

OOP + čisté PHP

Triedy pre DB, repository, autentifikáciu. Žiadny framework/CMS.

MySQL 8 + PDO

Prepared statements, správne typy, timestampy, migrácia cez SQL súbor.

CRUD článkov

Create/Read/Update/Delete aspoň nad entitou “writeup/article”.

📈Konkrétny postup (kroky)

  • 1 Upratať UI: jednotná navigácia, SENTRIX texty, odstrániť AI branding.
  • 2 Pridať DB schému: tabuľka users a articles.
  • 3 CRUD pre články (admin): create/edit/delete iba po prihlásení.
  • 4 Bezpečnosť: CSRF tokeny, output escaping, validácie, session settings.

Tip na obhajobu: ukáž, že rozumieš prečo používaš prepared statements (SQLi), prečo robíš escaping (XSS) a prečo je CSRF token nutný.

Ďalší krok: Dashboard (CRUD)

Po prihlásení správa článkov + neskôr komentáre a tagy.
